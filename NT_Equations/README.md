### What are Integer Equations?

*Integer Equations* are equations, where we are asked to find all integer (or positive integer) solutions. For example, it's when we are asked to find all positive integers (a,b,c,n), such that $a^n+b^n=c^n$.
As usual, there are 3 files about each problem: one describing the problem statement and my reasoning, second containing the program and third with the results.
