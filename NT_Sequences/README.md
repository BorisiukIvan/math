# Welcome to NT_Sequences!

### Please note that all work on sequences involving integer factorization is located in the "NT_Factoring" directory.

## What are integer sequences?

Integer sequences are the most interesting field in mathematics for beginners. For example, the Collatz's conjecture is the most unsolved popular problem in the world and it involves integer sequences.
If you want to see problems involving sequences of real numbers, you should see the chapter *AL_Sequences*.

As always, for each problem, there is a markdown file with its description, another file with program code and sometimes a third file containing results and progress.
